Windows 10文件类型关联的默认应用程序

发表于2025-04-01|更新于2025-04-01|Windows10某一文件格式无法通过双击打开，解决.msi文件无法安装的问题
|浏览量:

Windows10帮助论坛：Tenorumshttps://www.tenforums.com/

起因是不知道为什么我的电脑.msi文件无法通过双击进行安装了，msi文件的安装包还挺常见的。所以这个问题需要解决一下。

啥是MSI？ 直接百度百科：https://baike.baidu.com/item/MSI/10209

MSI（MSI =microsoft installer）文件是Windows Installer的数据，它实际上是一个数据库，包含安装一种产品所需要的信息和在很多安装情形下安装（和卸载）程序所需的指令和数据。MSI文件将程序的组成文件与功能关联起来。此外，它还包含有关安装过程本身的信息：如安装序列、目标文件夹路径、系统依赖项、安装选项和控制安装过程的属性。

通过搜索得知MSI文件是通过msiexec.exe进行安装的。

直接Win+R运行msiexec，就会弹出如下窗口，通过安装选项可以进行手动安装.msi文件.

直接在CMD窗口运行即可打开安装程序。

以上方法可以完成运行.msi文件，但是这不是我想要的，也不能每次都按照以上方法运行吧，继续搜索帮助。

然后就发现了一个网站，专门针对Windows10的一个论坛，里面都是讨论Windows10的问题，

然后我就发现了这篇文章：https://www.tenforums.com/tutorials/8703-restore-default-file-type-associations-windows-10-a.html

里面讲到如何在 Windows 10 中恢复文件类型关联的默认应用程序，其实.msi也属于一种文件类型嘛

双击不能进行安装就说明找不到Windows找不到要打开.msi文件的默认程序嘛。

我就用到了里面的方案二，下载了一些注册表文件。

直接下载压缩包：Windows 10 default file associations.zip

解压压缩包后直接双击对应的格式就恢复了Windows默认的应用程序，正好符合我的需求。

添加到注册表后尝试双击.msi格式文件，正常启动了安装程序。问题解决
