浏览器扩展插件

发表于2023-07-26|更新于2024-01-24
|阅读量:

Chrome 应用商店：

官网：Chrome 应用商店

Edge扩展插件

官网：Microsoft Edge 加载项

Tampermonkey

提到浏览器扩展必不可少的肯定是油猴

油猴的官网：https://www.tampermonkey.net/

可以直接去谷歌商店去下载安装 Tampermonkey

也可以在官网点击下载按钮进行下载安装

（ 另外我发现一个问题，如果你使用Edge访问官网，可能会没有以下下载按钮。）

油猴是非常强大的扩展插件，具有很高的执行权限。安装的时候一定要确认脚本的安全性，因为浏览器在使用中会获取很多平台的密码。各种各样的个人信息

虽然油猴的插件很方便，安装脚本的时候一定要注意脚本的安全性。（会看代码的去看一下源码）安全第一！

AdGuard

很好用的广告屏蔽插件，同样有手机软件和电脑软件，只不过是收费的。

AdGuard官网：https://adguard.com/zh_cn/welcome.html

Chrome 应用商店下载地址：AdGuard

Github项目地址：https://github.com/AdguardTeam/AdguardBrowserExtension

猫抓(cat catch)

猫抓(cat-catch) 资源嗅探扩展，能够抓取当前网页上面的音频视频文件。主要就是这个功能，也是在Github开源的项目，很老牌的一个扩展。

了解插件的更多信息，请前往帮助文档：https://o2bmm.gitbook.io/cat-catch/

Github开源地址：https://github.com/xifangczy/cat-catch

Chrome 应用商店下载地址：https://chrome.google.com/webstore/detail/jfedfbgedapdagkghmgibemcoggfppbb

IDM Integration Module

360安全浏览器：应用市场 (360.cn)

Edge扩展下载地址：IDM Integration Module

需要配合IDM客户端使用。

沉浸式翻译

很好用的网页翻译扩展，有些时候感觉没有Edge自带的好使。有些时候已经登陆了某些东西，不方便再复制到Edge访问可以用这个临时顶一下。其实也挺好用的，最大的特点是对照式翻译，原文和译文都显示。翻速度一般。了解更多请移步帮助文档

Github开源地址：Github沉浸式翻译

Chrome 应用商店下载地址：沉浸式翻译

帮助文档地址：https://immersivetranslate.com/docs/

WebRTC Control

谷歌应用商店：WebRTC Control
